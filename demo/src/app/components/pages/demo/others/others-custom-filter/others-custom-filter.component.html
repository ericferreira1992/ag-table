Custom filter

This example shows how to insert a custom filter. Maybe will need to follow the steps below.

  • Add #header on .
  • Add directive custom-filter="yourField" with the data field (more information this directive here).
  • Put your input inside .
  • Add the AgTable class input to style and maintain the fields pattern: class="ag-table-filter-input".
  • Set into your input the [formControl]="header.filterCtrls.yourField".
  • And finally, put on your input a *ngIf="header.filterCtrls.yourField" to check the filter control it's ready.
