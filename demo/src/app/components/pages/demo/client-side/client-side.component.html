Client side

This is used when all the data is on the client side or when it already all been returned from the server. This way you can to paginate, sort and filter all data direct in client side.

Example
ID Name Date Type Status {{item.id}} {{item.name}} {{item.dateRef | date:'yyyy/MM/dd'}} {{item.type}} {{item.active ? 'Active' : 'Inactive'}}
