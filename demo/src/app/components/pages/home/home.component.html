Porque usar?

O AgTable é uma excelente alternativa para o desenvolvedor que quer agilidade e praticidade na hora desenvolver. Com ele é possível montar uma grade de dados em poucos segundos, sem esforços. Com uma estrutura bem elaborada, não há necessidades de se criar compronentes para as colunas ou celulas, como alguns outros componentes de data-table exigem.

Porque ele é legal?

Os motivos para se usar o AgTable são diversos, alguns pontos o faz se destacar diante outros diversos data-tables existentes, veja quais:

  • Virtual scroll (com altura diferente das linhas)
  • Performance ao exibir milhares de registros
  • Filtro, ordenação e paginação no lado do clinte
  • Permite exibir dados obtidos pelo servidor (com inifity-scroll opicional)
  • Suporte para filtro, ordenação e paginação ao lado do servidor
  • Não obriga a criar componentes para customizar colunas e células
  • As medidas das colunas são customizaveis
  • E muito mais;
