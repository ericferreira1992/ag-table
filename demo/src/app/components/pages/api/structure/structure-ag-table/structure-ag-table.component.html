AgTable
ag-tableag-table/

Base component to initialize. Here you will need to put ag-table-header and ag-table-body

@Inputs
Name Type Default value Description
items any[] [] Data items
height string 'auto' That will define the data table height, and it will has scroll bar in case of overflow
clickable boolean false If the rows it will have click (just enable hover effect and cursor as pointer)
paginate number null This define if will go paginate or not
infinity number null This define if will go bring data based on scrolling when to arrive on bottom. (but use this only with server-side)
loading boolean false Show loading spinner above the data table when loading is true
data-length number 0 The total data length. This is used for make the pagination
server-side boolean false Determines whether data will be swallowed via server-side
get-data-init boolean false It will start by issuing a call in OnGetData so that it fetches your data (server-side only)
current-page number 1 Set the current page when use pagination
data-are-over boolean false Notifies that the data is gone (infinite-scroll only)
no-empty-view boolean false Sets whether to display empty-view or not
@Outputs
Name Response Type Emitted when
onGetData AgTableEvent When using the server side and when changing page, column filter and sorting. Or when using infinite-scroll.
onDataRender AgTableDataRenderEventT Emit event wich items have been rendered
onTableRender void Emit event when the ag-table has all been rendered
filterActivated boolean Emit event when the filter is active or not.
Public Methods/Properties
Name Return/Type Description
refreshRender() void Refresh the render rows.
resetFilters(getData: boolean = false) void Reset all filters. If getData is defined as true the onGetData is emitted.
header AgTableHeaderComponent The instance ofag-table-header/ (AgTableHeader)
body AgTableBodyComponent The instance of ag-table-body/ (AgTableBody)
