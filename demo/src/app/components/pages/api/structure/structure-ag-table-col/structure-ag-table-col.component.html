AgTableCol
ag-table-colag-table-col/

Component where you define the columns of your data grid. It is important to know that the number of columns should equal the number of cells inside the rows.

@Inputs
Name Type Default value Description
width string null Width of column. If not defined, this will be set automatically.
filter AgTableFilterType AgTableFilterType.TEXT When setting this, the column make a input based on defined value
Example:
.
custom-filter |
|
string |
string[] 		null This is used to make a custom filter. This is often used when the types of filters not apply for you.
field string null This is to be used when filtering or sorting.
mode AgTableFilterMode AgTableFilterMode.CONTAINS This is to define how the column will be filtered.
placeholder string '' This is the placeholder that will be shown in the filter input.
minDate Date null The minimum date of filter.
maxDate Date null The maximum date of filter.
dateFormat string 'yyyy/MM/dd' Date format that will be displayed on filtering.
dateSeparator string '/' Separator that will be displayed on date filtering.
optionAllLabel string 'All' Option displayed when no options of filter type of 'select' is selected.
options [] Selection options when used filter of type 'select'.
no-sort boolean false The columns has sorting as true per default. Puting this, the column is not will go sort.
no-truncate boolean false The columns has truncate text per default. Puting this, the text of column is not will go truncate.
sort string 'asc' The columns has sorting as true and ascendant per default. The column is will go sort based on value ('asc' or 'desc').
disable-filter boolean false This is disabled input of form control.
filter-value any null This is initial value of input filter.
